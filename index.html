Setup Guide

1 — Create a Google Cloud Project

Go to console.cloud.google.com → New Project → name it Budget JB.

2 — Enable APIs

APIs & Services → Enable APIs → enable Google Sheets API and Google Drive API.

3 — OAuth Consent Screen

APIs & Services → OAuth consent screen → External → App name Budget JB. Add scopes: spreadsheets and drive.file. Add your Gmail as a test user.

4 — Create OAuth Client ID

Credentials → Create Credentials → OAuth Client ID → Web application. Add your GitHub Pages URL to Authorised JavaScript origins. Copy the Client ID.

5 — Add your Client ID to index.html

Find YOUR_GOOGLE_CLIENT_ID in the file and replace it with your actual Client ID.

6 — Deploy to GitHub Pages

Create a repo budget-jb → upload index.html → Settings → Pages → Branch: main. Your URL: https://yourusername.github.io/budget-jb
